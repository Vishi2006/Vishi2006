# 💫 About Me:
I'm Pulkit, a third-year B.Tech Computer Science student at Jaipur Engineering College with a passion for building full-stack web applications. I came across this opportunity and felt it was a great fit for my skills and the kind of work I enjoy.<br>I've been working with the MERN stack — React, Node.js, Express, and MongoDB — and have built projects that reflect that. My projects include JWT authentication with bcrypt, database CRUD operations, responsive UI development, REST API integration, role-based access control, and full-stack application architecture using modern web technologies.<br>I also have knowledge of Python and regularly practice Data Structures and Algorithms in C++ to strengthen my problem-solving and logical thinking skills. <br>Additionally, I am exploring Web3, Web Animations, Mobile apps and  SaaS-based application development using Next.js, TypeScript, OAuth authentication.<br>I care about writing clean code and building things that feel good to use. I'm a quick learner and enjoy working collaboratively with a team.<br>
